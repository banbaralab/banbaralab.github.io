研究

Knowledge Representation and Reasoning (KRR)（知識表現と推論）とは，人工知能（AI）における基盤分野のひとつです．知識を形式的に表現し，それに基づいて論理的な推論を行うための理論と技術を扱います．

Answer Set Programming（ASP）は，知識表現と論理的推論のための宣言的プログラミングパラダイムである．問題の構造や制約を論理規則として記述することで，プログラムが自動的に解集合（answer sets）を計算し，それが問題の解に対応する．
SAT-based Constraint Programming（CP on SAT）は，CPによって宣言的に記述された組合せ問題を，SATに変換して解くアプローチである．CPの高い表現力とSATソルバーの高速性を組み合わせることで，効率的かつスケーラブルな問題解決を可能にする．
